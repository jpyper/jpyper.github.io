Welcome

Published on: 2025-06-09

Thank you for taking the time to visit my new little website. It is a new learning process for me.

My intentions for this site is to not only learn how to work with Markdown, plus its more modern flavors and a Static Site Generator (SSG) like Zola, but to also maintain this site with Git, setup some kind of automated job to push any changes to my GitHub repository so that they show up in the GitHub Pages and become a part of this conglomeration of bits and bobs that I call home now.

Be sure to check out the wiki section. I will be posting things there that I have written down or copied into a notes app that I think are useful. There I will be sharing those things with you.

I am still learning and how all of this flows together, so things could possibly break or not look right at times.

Check back on this blog once in a while and I will have updates of what is going on with me.
